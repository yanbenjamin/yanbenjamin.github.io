A Snapshot of My Digital Life

02 January 2023

Life Tech

A menagerie of digital tools and accessories I use (with varying degrees of frequency).

Coding
  • Google Colab: Go-to IDE for Python programming and ML-related stuff. Free access to Google Cloud GPUs is a massive plus.
  • Visual Studio Code: IDE for web development, Java, SQL, and managing local repositories. My favorite IDE for handling large codebases.
  • GitHub : Open source and version control at its finest.
  • Sublime Text: Another IDE I use for web development. Great colors and friendly interface.
  • Vim: A classic. Mainly for C / C++.
  • Homebrew: A must for package installation and management on macOS.
  • Conda: Building and integrating Python environments.
  • Docker: Containerizing and deploying applications. Go-to for virtualization and orchestration software.
Reading & Writing
  • Microsoft Word : Irreplacable for creative and professional writing.
  • Google Drive : For basically everything I type that isn't creative, professional, or code.
  • Overleaf: LaTeX is really useful for AI and algorithms classes, but I'm (admittedly) too lazy to use it when it's not required.
  • GoodNotes: Writing up problem sets for math, physics, and EE on iPad. Reading and annotating PDFs.
  • Notability: Scratch work for calculations and derivations on iPad. Sometimes, art.
  • OpenAI ChatGPT: Helps this introvert write emails.
Music
  • Spotify : Listening to songs and albums (Taylor Swift, Beach Boys, One Republic, Phoebe Bridgers).
  • YouTube : Listening to game (Zelda, Phoenix Wright, Skyrim) and movie (Interstellar, Harry Potter, Star Wars, Lord of the Rings) OSTs.
Movies & Shows
  • Netflix: Pixelated therapy. Highly recommend the recent Wednesday, Heartstopper, and Glass Onion: A Knives Out Mystery.
  • Disney+ : Time capsule to my most cherished movies (Pixar & Disney classics, Avengers).
Social Media & Communications
  • Instagram : Cat pictures & videos.
  • Slack : Work. Occassionaly, cat pictures.
  • Discord : Class groupchats. Occassionaly, cat pictures.
Games
  • Chess.com : I swear I'm getting better at the Sicilian, but I'm really not. I fancy playing the Guioco Pianissimo and Queen's Gambit (which I learned very recently is not just the name of a show).
  • Balloons Tower Defense 6: Best place for balloon popping and corporate micro-management.

There's also some tools I want to get around to using, but haven't yet.

  • Notion: For workflow planning.
  • GitHub Copilot: For AI-assisted programming as a VSCode extension.
  • Canva: More aesthetic creative output.
  • Tmux: Terminal multiplexer and running remote jobs.
  • GitLab: Built-in CI/CD pipelines and Jenkins.
  • Kubernetes: More software for containerized applications and virtualization.
  • The Most Dangerous Writing App: Now, we're talking. Interested to see if this can help cure my writer's block.
  • Final Draft: Always wanted to try screenwriting, but never really found the right time to start. Maybe it has a place in my 2023 resolutions?
  • Scrivener: Might be a cool place to write creative prose outside my comfort zone of the Microsoft Suite.
  • Grammarly: My montage of humiliating grammar and syntax mistakes keeps on growing. It may require intervention.
  • Duolingo: Learning more languages (Japanese, Italian), but mainly to enhance my Chinese and Spanish.

Happy to hear any suggestions for tools or ways to optimize my work stack!
